# NetSuite File Cabinet Skill

Upload and find files in NetSuite File Cabinet via the API Gateway.

## Quick Start

```bash
# Upload a file
python3 scripts/upload_file.py --file ./script.js --folder-id 137935 --env prod

# Find a file
python3 scripts/find_file.py --name "myScript.js" --env prod
```

See [SKILL.md](./SKILL.md) for full documentation.
